Metodologi

Alur Pemrosesan — Design Thinking

01
Empathize — Identifikasi Masalah
Wawancara dengan tim ONSA, identifikasi keterbatasan pemantauan konvensional di 334 km koridor RoW pipeline gas Arun–Belawan.
02
Define — Rumusan Masalah
ONSA membutuhkan metode pemantauan deformasi tanah yang sistematis, menyeluruh, dan berkelanjutan untuk seluruh panjang pipeline.
03
Ideate — Pemilihan Solusi
DInSAR berbasis Sentinel-1 dipilih karena cakupan 250 km, data gratis, resolusi 14 m, dan revisit 6–12 hari — optimal untuk pemantauan regional.
04
Prototype — Pengolahan Data
S1 TOPS Coregistration → Interferogram → Goldstein Filter → SNAPHU Unwrapping → Phase to Displacement → Python extraction & Jenks classification.
05
Test — Validasi Hasil
Validasi menggunakan NGL GNSS (stasiun SAMP), coherence spatial consistency check, dan perbandingan literatur subsiden Sumatera Utara.
06
Output — Peta Risiko Interaktif
3,348 titik HM terklasifikasi 8 tier Jenks Natural Breaks. Peta timeline interaktif dengan dual-thumb slider 3 periode dan filter risiko.
