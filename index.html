Metodologi

Alur Pemrosesan — Design Thinking

01
Empathize — Identifikasi Masalah
Wawancara dengan tim ONSA, identifikasi keterbatasan pemantauan konvensional di 334 km koridor RoW pipeline gas Arun–Belawan.
02
Define — Rumusan Masalah
ONSA membutuhkan metode pemantauan deformasi tanah yang sistematis, menyeluruh, dan berkelanjutan untuk seluruh panjang pipeline.
03
Ideate — Pemilihan Solusi
Dipilih DInSAR berbasis Sentinel-1 karena cakupan 250 km, data gratis, resolusi 14 m, dan revisit 12 hari — optimal untuk pemantauan regional.
04
Prototype — Pengolahan Data
S1 TOPS Coregistration → Interferogram → Goldstein Filter → SNAPHU Unwrapping → Phase to Displacement → Python extraction & APS correction.
05
Test — Validasi Hasil
Validasi menggunakan data NGL (Nevada Geodetic Lab) rata-rata dalam radius 20 km: InSAR −45.9 mm vs literatur −48.6 s.d. +54.1 mm/tahun untuk Sumatera Utara.
06
Output — Peta Risiko Interaktif
3,348 titik HM terklasifikasi dalam 8 tingkat risiko, divisualisasikan dalam peta Folium interaktif dan landing page ini sebagai deliverable operasional.
